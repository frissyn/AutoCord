Information

AutoCord is a manager for user-created Discord™ bots on the Repl.it server. Because the Repl.it Discord server is primarily a programming and coding server, we allow users to submit bots that they've created and invite them for learning and expirimentation. AutoCord exists to automate and organize the submission process, and to allow you to manage your submitted bots.

Submitting

In order to submit a Discord bot to the Repl.it server, you'll need to login to AutoCord with your Discord account, and create a new submission. You need to provide a few details:

  •  Your Bot's Name
  •  Your Bot's Prefix (i.e "c!")
  •  Your Bot's Permissions (This should be "0", contact Zwack for exceptions).
  •  Your Bot's Source Code (This is usually a REPL, but it can also be a GitHub repo. Also, if you aren't comfortable with sharing you bot's code, you can make it private again after getting your bot approved.)
  •  A brief description of what your bot does, and why you want to add it to the Repl.it server. Be creative! ;)

Here's a complete example of a Bot Submission:

 AutoCord

Bot Name: AutoCord

Bot Prefix: a&

Bot Permissions: 0 - Send Messages

Bot Description:

AutoCord is a bot created to help manage the submission of user bots within the Repl.it Discord Server!
